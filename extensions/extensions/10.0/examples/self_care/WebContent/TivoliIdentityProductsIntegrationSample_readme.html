IBM Tivoli Identity Manager Provisioning Fast Start Web Application Sample 5.1

About

The Web Application Sample is a functional web application that is geared towards end-users.  Features include:
  • Single Sign-on with WebSEAL
  • Challenge response (also known as "forgot password")
  • Self-registration
  • Self-care (allowing a user to manage his personal data)
  • Change password
  • Subscribe to applications 
    •

Documentation

Documentation for the Web Application Sample can be found in the IBM Tivoli Identity Manager Provisioning Fast Start Guide.  This guide is located with the product manuals for IBM Tivoli Access Manager for e-business version 5.1.
