Running the adhocreporting Example

Compiling and Installing the Example

  • Read the Relevant Reconciliation Database Tables and Queries.doc to understand the relevant database tables and queries that can be run to gather previous reconciliation data.
  • There is a zip file called HookedReconReportSample.zip which contains source code and instructions on how to build and configure the sample hooked reconciliation report. Look through the parts in this zip file to understand how the example works or to create your own custom reports.

Running the Example

To configure and run the example, follow the instructions in Hooked Reconciliation Report.doc which can be found in HookedReconReportSample.zip
