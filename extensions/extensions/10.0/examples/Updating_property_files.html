Updating the property files

1.    Log on to the IBM Security Identity Manager virtual appliance console to open the Appliance Dashboard.

2.      From the top-level menu of the Appliance Dashboard, select Configure > Advanced Configuration > Update Property to display the Update Property page.

3.    In the Update Property page, click the All properties tab.

4.    Click Identity server properties.

5.    Select the properties file.

6.    Select the property name in the property table. To search a property name, use the search box. Type a name or a character string to help you narrow your search.

7.    Click Edit.
