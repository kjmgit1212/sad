Steps to run the REST API examples

  1. Create a directory structure as shown in the following table.
    Table 1. Directory structure to run the REST API examples
    Directory Contents
    Sample The base directory that contains the /lib and /out directories.
    Sample/lib Contains libraries that are required to run the REST API examples.
    Sample/out Contains the compiled code.
  2. Add the following JAR files to the Sample/lib directory.
    • wink.1.x.jar. You can download the JAR file from https://wink.apache.org/
    • com.ibm.jaxrs.thinclient_8.5.0.jar . This JAR file is in the WebSphere Application Server installation directory WAS_HOME/runtimes
  3. Copy the ISIM_HOME/extensions/6.0/examples/rest_example_config.properties file to the Sample/out directory.
  4. Compile the code. Run the command from the Sample/out directory.
    Windows operating systems
    javac -classpath ../lib/com.ibm.jaxrs.thinclient_8.5.0.jar;../lib/wink-1.x.jar
ISIM_HOME/extensions/6.0/examples/restapi/examples/api/*.java
ISIM_HOME/extensions/6.0/examples/restapi/examples/util/*.java -d .
    UNIX or Linux operating systems
    javac -classpath ../lib/com.ibm.jaxrs.thinclient_8.5.0.jar:../lib/wink-1.x.jar 
ISIM_HOME/extensions/6.0/examples/restapi/examples/*/*.java -d
  5. Run the code.
    Windows operating systems
    java -classpath ../lib/com.ibm.jaxrs.thinclient_8.5.0.jar;../lib/wink-1.x.jar;./ examples.api.SearchPeople
    UNIX or Linux operating systems
    java -classpath ../lib/com.ibm.jaxrs.thinclient_8.5.0.jar:../lib/wink-1.x.jar:./ examples.api.SearchPeople
